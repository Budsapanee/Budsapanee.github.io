The Problem

Hardware programming is challenging. According to the survey, it is challenging in many ways; for example, each hardware item requires a distinct command mechanism depending on the manufacturer, and there is also an external environment to consider.

My Role

I collaborated with a software developer and a hardware developer to develop block diagrams and commands for our project. For the MakerPlayground project, I must design the user interface as well as the design system.

In addition, I must organize workshops for those who are interested in creating hardware projects. Then, do interviews to gather additional information from people who are truly interested in using it.

Challenge

The most challenging aspect is keeping the incomprehensible as simple as possible while complying to coding guidelines.

Solution & Features

According to the findings of the research, hardware programming should be event-based rather than line-based.
